## 2026-04-06 — Phase 1 完整实验 + Phase 2（2D地理）实现与探索

### 一、项目起源与设计哲学

项目起源于一个令人惊讶的数学事实：**500个财富完全相等的人，进行公平硬币翻转交换，反复数千次后，财富分布会自发涌现出指数分布（玻尔兹曼分布），基尼系数趋近0.5**。没有制度偏见，没有能力差异，纯粹是统计学的结果。这与气体分子能量分布在数学上完全同构。

框架设计核心原则：三层抽象（Agent / Environment / Simulation）、Matcher分层（Factor → Combiner → Selector）、Event机制、YAML配置驱动、单一随机源保证可复现性。

### 二、Phase 1：加性随机交换实验

#### 2.1 基础实验：指数分布的涌现

**设置**：500个agent，初始财富均为100，每次交换±1（delta=1），无吸收壁，5000轮。

**结果**：
- 财富分布从均匀分布逐渐涌现为指数分布
- 5000轮后 Gini ≈ 0.345，100000轮后 Gini ≈ 0.495（趋近理论值0.5）
- 均值始终守恒在100

**关键推论**：指数分布的参数完全由均值决定（λ = 1/mean），均值守恒意味着λ固定。这与气体分子能量分布同构——分子碰撞只是随机重分能量，最终的玻尔兹曼分布是最大熵状态。

**平衡时间推导**：系统达到平衡的特征时间尺度 T ≈ (mean_wealth / delta)²，与agent数量无关，只与财富/步长比有关。

#### 2.2 吸收壁实验（破产机制）

**设置**：引入BankruptcyWall——财富归零的agent永久退出系统。

**反直觉结果**：
- 有吸收壁的系统，Gini **反而更低**（约0.3 vs 无壁的0.5）
- 活着的人数量越来越少，总财富集中在少数幸存者手中

**解释**：这是**幸存者偏差**效应——穷人被淘汰出统计样本，只剩下相对富裕的人，表面上看"不平等减少"了，但实际上财富极度集中在少数人手中。Gini度量的是活人之间的不平等，忽略了已死亡的多数人。

**吸收时间理论**：存在唯一终产者的期望时间 ≈ N × (mean_wealth/delta)²，等价于多玩家赌徒破产问题。delta=100时，约1000轮内完成全部清洗。

#### 2.3 遍历性问题

**实验**：对比集体空间平均（某一时刻活人的平均财富）与个体时间平均（某个agent一生的平均财富）。

**结果**：
- 集体空间平均（只看活人）= 397
- 所有agent一生平均（含死亡期间）= 140

**结论**：系统不满足遍历性。GDP增长（集体平均上升）不等于个人受益（时间平均下降）。这与Ole Peters 2019年关于遍历性经济学的研究完全吻合——"平均收益"和"典型个体体验"是两个完全不同的东西。

#### 2.4 财富流动性实验

**实验**：追踪top 10%最富agent的身份随时间的变化。

**无吸收壁**：富人排名频繁洗牌，财富流动性高，但体系整体不平等程度维持在Gini≈0.5。

**有吸收壁**：初期因运气好而积累财富优势的agent，往往更容易活到最后。早期运气具有路径依赖性——先积累财富→赌注更大但相对安全→更难破产→更容易成为终产者。

#### 2.5 能力优势实验（10%的agent拥有55%胜率）

**设置**：500 agent中10%拥有55%胜率，其余50%。

**结果**：
- 1000轮后，能力优势agent占据top 10%财富的概率显著高于随机
- 有吸收壁时效果更显著

**关键推论（纠正了错误直觉）**：
- 短期：随机噪声按 sqrt(T) 增长，能力优势按线性 T 增长
- 因此能力优势永远会在足够长的时间内克服随机厄运
- 但在有限轮数内（现实人生），能力不保证胜出
- **信噪比**：所需轮数 T ≈ (sigma/advantage)²

#### 2.6 乘性交换实验

**Type A（输家比例型）**：stake = fraction × loser's wealth
- Gini ≈ 0.182，自稳定，不产生凝聚
- 机制：穷人输不多，富人输得多，天然再平衡

**Type B（yard-sale/最小值型）**：stake = fraction × min(w_A, w_B)
- Gini ≈ 0.949，极端凝聚
- 大多数人财富趋近于零但不破产（指数衰减的渐近线）
- 少数人获得几乎全部财富

**为什么称作yard-sale（车库甩卖）**：每次交易，双方都只拿出自己能拿出的最少那份——就像车库甩卖时，买家不会一次性掏出比卖家整个摊位还多的钱。这个看似"公平"的规则，实际上极度有利于富人。

**Yard-sale凝聚的根本机制（波动率损耗）**：
```
赢: w → w×(1+f) = w×1.2，log收益 = +0.182
输: w → w×(1-f) = w×0.8，log收益 = -0.223
50-50时期望log收益 = -0.020 < 0
→ 典型财富以指数速度衰减，即使期望值守恒
```
赢了50次输了50次，财富 = w₀ × 0.96^50 ≈ 0.13×w₀，缩水到13%。

#### 2.7 税收与再分配实验

**设置**：yard-sale模型+吸收壁，对所有agent征收flat tax（比例税），立即均匀再分配。

**税率扫描结果**：
- 0.1%平均税率：Gini从0.918降至0.616，且防止所有破产
- 0.1%仅对top 10%征税：效果约为flat tax的85-90%

**政治经济学意涵**：
- 税收本质上是给系统增加一个对抗波动率损耗的负反馈
- 但不改变规则（仍是yard-sale），只能延缓凝聚而非阻止
- 对富人单独征税效率几乎与全员税相同，因为富人拥有绝大多数财富

### 三、Phase 2：2D地理空间实验

#### 3.1 设计讨论与参数确定

通过苏格拉底式对话逐步确定设计参数：

- **空间**：200×200离散torus网格（上下左右边界循环），500 agent，密度极低（1/80格）
- **移动**：每tick八方向随机游走（含原地不动），9选1均匀随机
- **多占格**：允许多agent共占同一格
- **匹配半径**：R=5（11×11=121格窗口），期望邻居数≈1.5，配对概率≈78%
- **匹配规则**：窗口内均匀随机选择（无权重）
- **可视化**：上方地图（散点图，颜色=财富，inferno色图）+ 下方财富直方图

**为什么选R=5**：
- R=1时期望邻居0.11，90%的agent每轮找不到对手
- R=10时几乎等同于随机配对，地理失去意义
- R=5时配对率约78%，地理有实质影响又不过于稀疏

**为什么用torus**：消除边界效应，空间均质，角落和中心agent具有相同的邻居数期望值。

#### 3.2 新增组件

- `Agent.grid_pos: tuple[int,int] | None`：2D网格坐标
- `Grid2DSelector`：在torus上的(2R+1)²窗口内均匀随机选partner
- `GridMovementEvent`：每tick移动，9方向均匀随机，坐标模grid_size
- `GridMovementEvent`的torus边界过滤：绿色连线只画给视觉上真正相邻的配对（坐标差≤grid_size/2），避免跨整个地图的长线段
- `viz/animate_2d.py`：matplotlib LineCollection画绿色配对连线，inferno色图散点图

#### 3.3 实验1：加性交换 + 2D地理 + 无吸收壁（delta=10）

**结果**：
- 1000轮后Gini = 0.527
- 每tick约160对配对（500人中64%找到对手）
- 均值守恒在100

**对比无地理约束**：全随机1000轮Gini约0.25+，地理约束下Gini反而更高？

**解释**：delta=10时系统更快收敛。地理约束每轮只有64%agent参与交换（而非接近100%），等效于每tick有效交互数减少，但每次交互金额更大（delta=10），系统收敛速度反而可比。

#### 3.4 实验2：加性交换 + 吸收壁

**结果**：
- 500→178个agent存活（淘汰64%）
- 均值从100→280.9（财富集中在幸存者）
- Gini = 0.303（低于无壁的0.527）
- 配对数从110降到32（地图越来越稀疏）

**关键观察**：agent越来越少，幸存者越来越难找到邻居，形成地理上的"孤岛"效应——活着的人彼此相距越来越远，交互频率持续下降。

#### 3.5 实验3：Yard-sale + 吸收壁 + 随机游走

**结果**：
- 500人**全部存活**（吸收壁从未触发！）
- Gini = 0.886
- 地图上：几十个亮点（富人）均匀散布，大量黑点（穷人趋近于零）

**为什么吸收壁无效**：
Yard-sale中stake = fraction × min(w_A, w_B)。当某人财富趋近0时，赌注也趋近0。财富以指数函数渐近趋近零，永远不会真正到达零。这是一个数学意义上的**渐近线**，吸收壁的触发条件（wealth ≤ 0）永远不满足。

**Socratic讨论推出的两层机制**：

**机制一（波动率损耗，已知）**：乘性随机游走中，典型财富指数衰减，总财富守恒，财富被动堆积到少数幸运者。

**机制二（财富屏障，对话中发现）**：
- 系统后期，90%财富在10%的人手里
- 富人随机游走，遇到的大概率是穷人
- 与穷人相遇时stake = fraction × 穷人财富，极小
- 富人财富几乎静止不动
- 穷人之间继续互相消耗，继续向零漂移
- **正反馈**：越富→遇到穷人概率越高→赌注越小→财富越稳定→越富

**与友谊悖论的联系（用户自发发现）**：
富人在系统中遭遇的邻居，是对整体人口分布的有偏采样（穷人占多数→采样到穷人概率高）。这与友谊悖论同构：你的朋友比你的朋友更多，因为受欢迎的人出现在更多人的朋友列表里，同样是一种"高值个体被过采样"的结构性偏差。

#### 3.6 实验4：Yard-sale + 固定位置（无随机游走）

**结果**：
- Gini = 0.615（比随机游走的0.886显著低）
- 富人岛位置**固定**，空间均匀分布
- 配对数稳定在~157对/tick

**"富人岛"空间结构的推导**：

固定位置时，每个agent的邻居集合固定为其出生位置的(2R+1)²邻域。如果一个邻域里有两个富人，他们会持续相遇，高风险博弈（**"黑洞吞并"**——双方都拿出大额赌注，其中一个被消耗至贫穷），最终一个胜出，一个归零。

因此：**每个R=5的邻域最多稳定存在1个富人**。这形成了类似细胞铺砖的空间结构——每个"细胞"（约11×11格的领地）有一个富人，周围被穷人包围。这就是**竞争排除原理**：占据同一生态位（同一邻域）的两个富人无法长期共存。

**为什么固定位置Gini反而更低**：
- 固定时：每个领地一个富人，富人数量稳定在 grid_area / window_area = 40000/121 ≈ 330个（理论上限），实际约几十个富人同时稳定存活
- 随机游走时：富人可以相遇（"黑洞吞并"），全局竞争进一步减少富人数量，财富更极端地集中在更少的人手里

**黑洞比喻（对话中涌现）**：
- 两个黑洞靠近 → 引力波/能量耗散 → 合并为一个更大的黑洞
- 两个富人靠近 → 高风险博弈 → 一个吃掉另一个，财富进一步集中
- 靠太近的富人对被系统自然淘汰一个
- 留下来的"岛"天然是分散的——不是主动回避，而是靠近的代价太高

#### 3.7 实验5：穷人优势梯度实验（2%~16%）

**机制**：当穷人遇到富人时，给穷人额外的win probability boost。

**数学临界点推导**：
fraction=0.2时，要使穷人财富不衰减，需要：
```
p × ln(1.2) + (1-p) × ln(0.8) ≥ 0
p × 0.182 + (1-p) × (-0.223) = 0
p ≈ 0.55（即5%优势）
```
**55%胜率是波动率损耗的盈亏平衡点**。

**梯度实验结果**：

| 穷人优势 | 胜率 | Gini |
|---------|------|------|
| 0% | 50% | 0.886 |
| 2% | 52% | 0.858 |
| 4% | 54% | 0.793 |
| 6% | 56% | 0.720 |
| 8% | 58% | 0.596 |
| 10% | 60% | 0.490 |
| 12% | 62% | 0.419 |
| 14% | 64% | 0.352 |
| 16% | 66% | 0.309 |

**关键观察**：
1. 数学预测临界点5%，实际拐点在8-10%——因为富人的"采样保护"（大概率遇穷人）额外加强了富人优势，需要更高的穷人胜率才能真正逆转
2. 10%优势（60%胜率）是重要阈值：Gini跌破0.5，从极度不平等进入中度不平等
3. 曲线非线性：6%→8%的Gini降幅（0.720→0.596）远大于0%→2%（0.886→0.858）

**现实对应**：累进税率、最低工资、社会保障本质上都是给系统注入`poor_advantage`参数——通过制度性规则，在每次"交易"中给弱势方额外优势，对抗yard-sale凝聚的两层机制。

### 四、核心推论总结

#### 推论1：财富不平等有两个层次
- **统计基线**（加性交换）：纯随机产生Gini=0.5的指数分布，无需任何制度偏见
- **凝聚机制**（乘性交换/yard-sale）：波动率损耗+财富屏障，将Gini推向0.9+

#### 推论2：波动率损耗是乘性系统的隐形税
50%赢率×50%输率的乘性游戏，每个人的典型财富都在衰减。这笔"税"不归任何人收，但财富守恒意味着它最终堆积到少数幸运者手中。

#### 推论3：财富保护是一个正反馈机制
一旦领先→遇到更弱对手→赌注更小→财富更稳定→进一步领先。这与友谊悖论同构：有偏采样本身就是一种结构性优势。

#### 推论4：地理约束减慢了财富集中的速度
有地理限制时，富人的"势力范围"被空间隔离，减少了全局竞争（"黑洞吞并"），维持了更多富人同时存在，Gini更低（0.615 vs 0.886）。

#### 推论5：制度干预需要超过临界阈值才有显著效果
5%穷人优势仅能抵消波动率损耗（数学临界点），不能克服采样保护带来的额外富人优势。需要约10%的优势才能将系统推入质变（Gini<0.5）。

#### 推论6：幸存者偏差会系统性低估不平等
有吸收壁时Gini更低，并非系统更公平，而是穷人从统计样本中消失了。任何只统计"活跃参与者"的不平等指标，都会系统性低估真实的不平等程度。

### 八、Smart Agent 机制实验

**设计**：随机10%的agent拥有`can_refuse=True`属性——当候选配对对手比自己富时，拒绝交易，本轮跳过。胜率恢复50-50（无穷人优势）。

**实现**：
- `Agent.can_refuse: bool = False`（新属性）
- `Grid2DSelector`：筛选候选人时，若`a.can_refuse and b.wealth() > a.wealth()`则跳过该候选
- `run_2d.py`：读取`smart_fraction`配置，随机分配`can_refuse`
- 配置：`configs/phase2_geo2d.yaml`加`smart_fraction: 0.1`

**smart比例梯度扫描（500 agents，1000轮）**：

| smart比例 | Gini | smart均值 | normal均值 |
|----------|------|----------|-----------|
| 0% | 0.886 | — | 100.0 |
| 10% | 0.875 | 39.9 | 106.7 |
| 30% | 0.882 | 122.4 | 90.4 |
| 50% | 0.887 | 104.5 | 95.5 |
| 100% | 0.883 | 100.0 | — |

**反直觉发现：10%的smart agent均值只有39.9，远低于normal的106.7！**

**Socratic推导的解释（双重机制）**：

**机制一（切断暴涨）**：
Smart agent拒绝富人 → 主要和穷人交易 → stake = fraction × 穷人财富 → 极小
→ 低波动率损耗，但也无法通过击败富人暴涨
→ 财富被困在低水平震荡

**机制二（30%时反转）**：
当smart比例升至30%，富人找不到足够的smart对手，只能集中捕食70%的normal agent
→ Normal group承受不成比例的富人捕食压力
→ Smart group相互低风险交易，形成"中产保护圈"
→ Smart均值122.4 > Normal均值90.4

**关键推论**：Gini在各smart比例下几乎不变（约0.88）——整体不平等程度不变，只是**不平等的受害者在smart和normal之间重新分配**。

### 十四、Chase Agent（追财富方向移动）

**机制设计**：10%的agent拥有特殊移动策略——根据上一轮匹配结果决定方向：
- 上轮对手更富 → 朝对手方向移动1步（追富人）
- 上轮对手更穷 → 向反方向移动1步（远离穷人）
- 上轮财富相等或无记忆（第1轮） → 随机游走
- 方向计算：`sign(Δrow), sign(Δcol)`，自然支持对角线，计算最廉价

**实现**：
- `Agent.chase_wealth: bool`、`_last_partner_pos`、`_last_partner_wealth` 新属性
- `GridMovementEvent` 内部区分 chase agent 和普通 agent
- 每轮 do_interactions 后更新 partner 记忆

**关键发现（基于矿区+无吸收壁系统，delta=5，5000轮）**：

| 指标 | Chase(10%) | Normal |
|------|-----------|--------|
| n（存活） | 46 | 356 |
| mean | 1160 | 858 |
| median | 1069 | 553 |
| max | 2888 | 4472 |
| cv | 0.62 | 0.94 |
| Mann-Whitney p | **0.0007（显著）** | — |
| 矿区停留比例 | **8.3%** | 5.0% |
| 随机基线 | 7.1% | 7.1% |

**空间机制验证**：chase agent 追富人 → 富人因矿区+1/tick 更富，倾向聚集矿区附近 → chase agent 被间接引向矿区（8.3% > 理论随机基线7.1%）→ 5k轮后财富显著更高。**chase收益不来自交易本身，而是纯空间效应**。

**1k轮时不显著（p=0.37）→ 5k轮才显著（p=0.0007）**：需要足够多轮次让空间优势积累。

**为什么normal的max更高**：
1. 样本量：9:1的normal/chase比例使normal有更多极端值机会（次序统计量效应）
2. Bootstrap验证：从normal等量抽148人取max，10000次模拟均高于chase_max（p=1.00），确认max差异在控制样本量后依然显著
3. 根本原因：chase策略压缩方差（cv 0.62 vs 0.94），低方差分布天然产生更薄的尾部

**Chase 30%实验**：
- Chase矿区停留率18.3%（远超理论7.1%），形成正反馈聚集
- Normal mean反而上升（917 vs 858）：chase大量占据矿区，normal被"挤进"矿区溢出效应受益
- 差异更显著（p<0.0001）

### 十五、Smart Agent 单边性讨论

回顾之前smart agent（can_refuse）的实现：拒绝逻辑只在"主动方"（先被遍历到的agent a）起作用。若rich normal先遍历到poor smart，rich会把poor smart加入候选并选中——交易仍然发生。因此smart保护概率约50%，不是100%。两种语义：
- 当前实现：可拒绝别人邀约，但也可被动被选中
- 双边检查：真正的双向同意才交易，保护更强

### 十六、价格自发形成市场

**新系统设计（Socratic对话逐步确定）**：
- 移除矿区、移除吸收壁（改为饥饿壁）
- 两种资产：coin（通货）和nec（必需品）
- 全域coin和nec守恒

**参数确定过程**：
1. 消耗机制：强制以任何价格购买（无法购买时死亡）
2. 价格形成：影子价格中点（shadow_price = coin_i / nec_i）
3. 初始禀赋：两者均等（coin=100, nec=50）
4. 卖家无surplus时跳过配对
5. 农田生产：空间圆形patch，各拿固定量（等额分享）
6. 原随机通货交换去掉（纯市场驱动）
7. Tick顺序：移动 → 消耗+生产（原子操作） → 饥饿壁 → 配对 → 交易

**农田设计**：
- 5个圆形农田，半径R=30，固定位置（骰子5点布局）
  - (50,50), (50,150), (100,100), (150,50), (150,150)
- 覆盖35.3%地图，期望353人常驻农田
- 理论均衡价格 = total_coin / total_nec = 100,000/50,000 = 2.0 coin/nec

**新增代码**：
- `NecessityLifecycleEvent`（原子守恒事件）：
  1. 所有agent消耗 min(nec, rate) 并记录总量
  2. 将总消耗量均分给当前在农田上的agents
  → 完美守恒：每tick Σnec不变
- `StarvationWall`：nec=0则死亡，死时coin均分给存活者（保证coin守恒）
- `NecessityTradeInteraction`：
  - 影子价格高的为买家，低的为卖家
  - 成交价 = 两者影子价格中点
  - 饥饿买家（nec=0）接受卖家影子价格
  - 成交量 = min(seller_surplus, buyer_can_afford)
  - 浮点安全：coin_paid = min(quantity×price, coin_buyer)

**调试过程记录**：
1. Bug 1（nec不守恒）：饥饿agent实际消耗0但按N×c计算产量，每个饥饿agent贡献净+1 nec/tick → 解决：原子事件先消耗后按实际量生产
2. Bug 2（coin不守恒）：浮点精度导致remove失败但add成功 → 解决：coin_paid = min(quantity×price, coin_buyer)

**实验结果（有饥饿壁，1000轮）**：

| tick | agents | mean_coin | mean_nec | nec_total | coin_total |
|------|--------|-----------|----------|-----------|------------|
| 100  | ~330   | 305       | 152      | 50000     | 100000     |
| 500  | ~141   | 709       | 355      | 50000     | 100000     |
| 1000 | 70     | 1429      | 714      | 50000     | 100000     |

**关键涌现结果**：

1. **守恒完美**：coin和nec全程精确守恒（死亡资产重分配机制生效）

2. **幸存者偏差驱动均值上涨**：死者资产均分给存活者，活人越来越富，均值线性增长

3. **价格自发锁定均衡**：`mean_coin / mean_nec ≈ 2.0` 在任何时间点成立
   - tick 200: 400/200 = 2.0
   - tick 700: 990/495 = 2.0
   - tick 1000: 1429/714 = 2.0
   - **去中心化双边交易自发涌现并维持了Walrasian均衡价格**

4. **价格波动大但均值偏高**：后200轮均价2.973（理论2.0），std=3.49——饥饿买家影子价格=∞，推高了成交均价；但系统资产比例保持在均衡值

5. **可视化**：4面板动画（coin空间分布、nec空间分布、价格时序+均衡线、nec直方图）

**待探索方向**：

- 初始nec量对存活率的影响
- 观察价格长期收敛行为（更多轮次）
